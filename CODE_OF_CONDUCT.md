# Code of Conduct for [CleanComms]

## The Rule
Don't be a dick without cause. We're building software, not saving the world from microaggressions.

## What That Means
- **Focus on the Code**: Contribute good shit, review honestly. If someone's code sucks, say so constructively.
- **No Unnecessary Drama**: Personal attacks, harassment, or whining? GTFO. But if someone's being an idiot, call them out.
- **Handle Your Own Shit**: Adults here. If you can't take criticism, this ain't for you.
- **Off-Topic? Nope**: Keep politics, feelings, and bullshit elsewhere.

## Enforcement
Report real issues to maintainers. We'll judge if you're just being a pussy or if it's legit. Assholes get banned, no therapy sessions.

Adapt if you want, but keep it simple.

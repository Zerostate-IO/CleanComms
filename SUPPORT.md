# Support Guide

CleanComms is a project of the ZeroState-IO organization. We provide several channels for support and community interaction.

## Support Channels

### GitHub Discussions (Q&A)
For general questions, how-to's, and community discussions, please use [GitHub Discussions](https://github.com/ZeroState-IO/CleanComms/discussions). This is the best place to get help from the community and the maintainers.

### GitHub Issues (Bugs)
If you have found a bug, please report it through [GitHub Issues](https://github.com/ZeroState-IO/CleanComms/issues). Before creating a new issue, please search existing issues to see if the bug has already been reported.

### Security Vulnerabilities
**Do not report security vulnerabilities through public GitHub issues.** Please refer to our [Security Policy](SECURITY.md) for instructions on how to report vulnerabilities privately.

## Community Health
- Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md).
- For contribution guidelines, please refer to our [Contributing Guide](CONTRIBUTING.md).

## Volunteer SLA
CleanComms is maintained by volunteers. We aim for a **5 business days first response target**, but this is not guaranteed. We appreciate your patience.

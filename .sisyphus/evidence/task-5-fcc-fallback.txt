### Provider Support Logic
- **HamQTH**: Supports all callsigns (worldwide)
- **FCC**: Only supports US callsigns (K*, W*, N*, AA-AL)

### US Callsign Detection
```go
func isUSCallsign(callsign string) bool {
    // US callsigns: K*, W*, N*, A*, AA-AL, KA-KZ, NA-NZ, WA-WZ
    prefix := callsign[:1]
    switch prefix {
    case "K", "W", "N":
        return true
    case "A":
        // AA-AL are US amateur callsigns
        second := callsign[1]
        return second >= 'A' && second <= 'L'
    }
    return false
}
```

### Fallback Behavior
1. HamQTH is always tried first (if credentials configured)
2. If HamQTH fails with ErrProviderDown, FCC is tried
3. If HamQTH returns ErrNotFound, lookup fails immediately (not found in primary)
4. FCC is skipped for non-US callsigns
5. Errors are logged but not exposed to caller (only final result)

### Cache Behavior
- Successful lookups are cached for configurable TTL (default 24h)
- Errors are NOT cached (subsequent calls retry provider)
- Cache keys are case-insensitive
- Expired entries are pruned on demand

## API Summary

### Client Functions
- `NewClient(host string, port int) *Client` - Create client instance
- `Connect() error` - Establish TCP connection
- `Disconnect() error` - Close connection
- `IsConnected() bool` - Check connection state
- `SetTimeout(d time.Duration)` - Configure timeout
- `Ping() error` - Health check via chk_vfo
- `GetFrequency() (int, error)` - Get VFO frequency in Hz
- `SetFrequency(hz int) error` - Set VFO frequency
- `GetMode() (string, error)` - Get current mode (USB, LSB, etc.)
- `SetMode(mode string) error` - Set mode with default passband
- `GetPTT() (bool, error)` - Get PTT state (true=TX, false=RX)
- `SetPTT(tx bool) error` - Set PTT state

### Error Types
- `ErrNotConnected` - Client not connected
- `ErrConnectionRefused` - rigctld not running
- `ErrTimeout` - Operation timed out
- `ErrMalformedResponse` - Invalid response format
- `ErrCommandFailed` - rigctld returned error code
- `CommandError` - Wrapped error with command context
